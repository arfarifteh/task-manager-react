# UX Mockups

This directory contains UX design mockups and visual references for the Task Manager application.

## Files

- `task-manager-dashboard-mockup-design.png` - Main application mockup showing the complete UI design

## Usage

These mockups are for documentation and reference purposes only. They are not bundled with the application.

To view the mockup, open the PNG file directly or reference it in documentation:

```markdown
![UX Mockup](./task-manager-dashboard-mockup-design.png)
```
